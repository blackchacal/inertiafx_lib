\doxysubsection{Detailed Description}
Represents a generic physical entity in the simulation world. 

The \doxylink{class_inertia_f_x_1_1_core_1_1_engine_1_1_i_entity}{IEntity} interface declares the core functionality required for all physical objects. These include position, velocity, acceleration, force, mass, and volume.
