\doxysubsection{Detailed Description}
Interface representing a physical medium of a certain medium. 

This interface provides methods to manage a physical medium\textquotesingle{}s properties, such as volume and mass. It allows for unit-\/aware types as well as overloads that accept scalar values combined with a specified decimal prefix.
