\doxysubsection{Detailed Description}
An abstract class representing a rigid (solid) physical body in the simulation. 

Inherits from \doxylink{class_inertia_f_x_1_1_core_1_1_engine_1_1_entity}{Entity} and adds density, a concrete shape interface, and a flag to mark the body as fixed (immovable).
