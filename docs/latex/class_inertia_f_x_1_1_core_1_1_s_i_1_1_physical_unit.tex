\doxysubsection{Detailed Description}
A base class that implements the \doxylink{class_inertia_f_x_1_1_core_1_1_s_i_1_1_i_physical_unit}{IPhysical\+Unit} interface. 

This class provides a generic implementation for a physical unit by storing basic attributes such as name, symbol, and description. It also provides string representations of the unit with or without decimal prefixes.
