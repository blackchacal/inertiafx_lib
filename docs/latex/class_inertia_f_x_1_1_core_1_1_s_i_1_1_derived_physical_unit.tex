\doxysubsection{Detailed Description}
A class to represent derived physical units that implements the \doxylink{class_inertia_f_x_1_1_core_1_1_s_i_1_1_i_physical_unit}{IPhysical\+Unit} interface. 

This class provides a generic implementation for a derived physical unit by obtaining the basic attributes such as name, symbol through a combination of the names and symbols of the base units that form the derived unit.
