Detailed Description

template<typename T>
class InertiaFX::Core::SI::DerivedQty< T >

Represents a derived physical quantity in the system (e.g., velocity, acceleration).

Template Parameters
TNumeric type used to store the value of the physical quantity.

This class implements only a subset of the IPhysicalQty interface:

  • getName()
  • getSymbol()
  • getDescription()
  • getUnitName()
  • getUnitSymbol()
  • getUnitDescription()
  • isFundamental()

All other methods remain pure virtual for further specialized classes.
